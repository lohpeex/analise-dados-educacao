# Análise de Dados na Educação Pública

## Descrição
Este projeto apresenta uma proposta de análise de dados voltada à educação pública, com foco na utilização de informações para apoiar a tomada de decisão.

## Objetivo
Demonstrar como dados escolares podem ser utilizados para melhorar a gestão educacional e o acompanhamento dos alunos.

## Possíveis análises
- Frequência dos alunos
- Desempenho escolar
- Taxas de evasão
- Comparação entre turmas

## Ferramentas (proposta)
- Excel
- Power BI
- SQL

## Contexto
A análise de dados na educação pública é essencial para identificar problemas, melhorar processos e apoiar decisões estratégicas.
